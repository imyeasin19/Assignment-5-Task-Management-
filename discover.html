Question-1: What are the different ways to select an element in the DOM?

You can select element in the DOM by

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerText return or set any text to an element
innerHTML return or set the HTML content of an element
textContent return or set the text content of an element

Question-3: What is event delegation in the DOM?

In DOM event delegation is a Javascript design pattern which efficiently handle events where you attach a single event listener to a parent element.

Question-4: What is event bubbling in the DOM?

In the DOM event bubbling is the behavior where when an event triggering event handlers on its parent elements until it reaches the root of the document

Question-5: How do you create, add, and remove elements using JavaScript?

To add, remove, and modify HTML elements using JavaScript, we can use methods like createElement, appendChild, removeChild, and properties like innerHTML.
